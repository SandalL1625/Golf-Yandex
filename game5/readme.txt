 Гольф - это игра, созданная как проект для Академии Яндекс Лицея По теме Pygame.

 Игра состоит из уровней, к которым можно попасть через меню игры.
 Нужно бить по мячу и преодолевать препятствия, чтобы добраться до лунки и пройти уровень.
 По мячу можно бить только когда он почти остановился.
 Для симуляции физики используется библиотека Pymunk.
 В проекте используется система тайлов для построения уровней. Уровни задаются в .txt файле в особом формате.
 Игра также использует систему сохранений для сохранения наилучшего результата.
 В меню можно менять мяч, громкость звука, запускать уровни и выходить из игры.
 В игре есть музыка. Как во время игры, так и в меню. Всего семь треков.
